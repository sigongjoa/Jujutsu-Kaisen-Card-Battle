## 2. 핵심 비전 성명서

### 2.1 한 문장 정의

> **"주술회전 세계관 속에서 저주술사들의 능력과 주술을 활용하여 전략적 카드 배틀을 펼치는, 진입 장벽은 낮지만 깊이 있는 TCG"**

### 2.2 확장된 비전

주술회전 카드 배틀은 다음의 경험을 제공합니다:

1. **IP 충실성** (Authenticity)
   - 애니메이션 스토리라인과 일관성 있는 카드 디자인
   - 캐릭터 능력이 게임 메커니즘으로 충실하게 구현됨
   - 팬들이 "이 카드는 이 캐릭터답다"고 느끼는 디자인

2. **낮은 진입장벽** (Accessibility)
   - 5~10분 내에 기본 규칙 습득 가능
   - 스타터 덱으로 즉시 플레이 가능
   - 직관적인 카드 텍스트와 시각적 명확성

3. **전략적 깊이** (Strategic Depth)
   - 덱 구성에 따른 다양한 전술 존재
   - 운과 기술의 균형 (약 60% 기술, 40% 운)
   - 매 패치마다 새로운 메타게임 형성

4. **커뮤니티 중심** (Community-Driven)
   - 토너먼트 및 조직화된 플레이 활성화
   - 사용자 피드백 기반 지속적 밸런싱
   - 프로 플레이어와 캐주얼 플레이어 모두 즐길 수 있는 구조

## 3. 차별화 요소 (Unique Selling Points)

### 3.1 경쟁사 분석

| 게임 | 특징 | 우리의 차별성 |
|------|------|------------|
| **매직 더 개더링** | 깊은 전략성, 높은 복잡도 | 더 단순하고 빠른 게임플레이 |
| **포켓몬 TCG** | 강력한 IP, 캐주얼 친화 | 주술회전의 독특한 전투 시스템 |
| **하스스톤** | 디지털 중심, 빠른 게임 | 물리적 카드 + 디지털의 하이브리드 |
| **유희왕** | 높은 파워 크리프, 복잡한 규칙 | 명확한 밸런싱과 지속적 조정 |

### 3.2 주술회전 TCG의 차별화 포인트

1. **3-Turn Death Match 포맷**
   - 3턴 이내에 결판이 나는 초고속 게임 (평균 10분)
   - 높은 운 요소와 블러핑 요소 포함
   - 초보자도 경험자와 승률 경쟁 가능

2. **주술(Cursed Technique) 시스템**
   - 캐릭터마다 독특한 주술 능력
   - 주술을 조합하여 콤보 생성
   - IP와 게임 메커니즘의 완벽한 일치

3. **저주력(Cursed Energy) 자원 시스템**
   - 매 턴 자동으로 증가하는 자원 (하스스톤식)
   - 초반/중반/후반의 명확한 템포
   - 덱 구성으로 자원 회복 가능

4. **하이브리드 플랫폼**
   - 물리적 카드로 오프라인 토너먼트
   - 디지털 앱으로 온라인 매칭
   - 양쪽 플랫폼 간 카드 수집 연동

## 4. 타겟 오디언스 분석

### 4.1 주요 플레이어 세그먼트

#### 세그먼트 1: 주술회전 팬 (40%)
- **연령**: 12~25세
- **특징**: IP 충실성을 최우선으로 함, 강한 팬심
- **플레이 스타일**: 캐주얼 ~중급
- **구매 의향**: 높음 (덱 테마 기반 구매)

#### 세그먼트 2: TCG 경험자 (35%)
- **연령**: 20~40세
- **특징**: 게임의 전략성과 밸런스에 주목, 경쟁심 강함
- **플레이 스타일**: 중급 ~ 상급
- **구매 의향**: 높음 (메타 덱 기반 구매)

#### 세그먼트 3: 신규 유저 (25%)
- **연령**: 15~35세
- **특징**: 친구의 추천으로 입문, 커뮤니티 중심
- **플레이 스타일**: 캐주얼
- **구매 의향**: 중간 (진입 비용 민감)

### 4.2 플레이어 경험 피라미드

```
              Pro Players (1%)
              ├─ 토너먼트 상위권 플레이어
              └─ 온라인 랭크 1000위 이내

           Competitive (14%)
           ├─ 정기적 토너먼트 참가
           └─ 덱 최적화에 관심

        Casual Regulars (35%)
        ├─ 주 1~2회 플레이
        └─ 커뮤니티 활동

     Very Casual / Collectors (50%)
     ├─ 월 1회 이하 플레이
     └─ 수집 목적 구매

목표: 모든 세그먼트를 모두 만족시킬 수 있는 게임 설계
```

## 5. 게임 경험의 핵심 가치

### 5.1 3가지 핵심 기둥

```
┌─────────────────────────────────────────────┐
│  Jujutsu Kaisen Card Battle - 핵심 가치    │
├─────────────────────────────────────────────┤
│                                             │
│  1️⃣  AUTHENTICITY (IP 충실성)              │
│      └─ 캐릭터와 능력이 게임에 살아남       │
│                                             │
│  2️⃣  ACCESSIBILITY (진입 용이성)           │
│      └─ 5분 안에 배우고, 10분 안에 플레이  │
│                                             │
│  3️⃣  STRATEGIC DEPTH (전략적 깊이)        │
│      └─ 카드 조합으로 무한한 전술 가능     │
│                                             │
└─────────────────────────────────────────────┘
```

### 5.2 플레이 경험의 순서

1. **덱 빌딩 (30분)**: 좋아하는 캐릭터와 주술로 덱 구성
2. **게임 시작 (1분)**: 직관적인 세팅
3. **게임플레이 (10분)**: 빠르고 긴장감 있는 전투
4. **승패 분석 (5분)**: "왜 졌는가?" 분석과 피드백
5. **다시 플레이**: 개선된 덱으로 재도전

## 6. 게임의 톤과 분위기 (Tone & Atmosphere)

### 6.1 비주얼 톤
- **어둡고 세련된 다크 판타지**
- 저주와 주술이 교차하는 긴장감
- 캐릭터들의 개성있는 주술 표현

### 6.2 게임플레이 톤
- **빠르고 역동적인 전투**
- 작은 실수가 승패를 결정하는 긴장감
- 블러핑과 읽기 플레이의 중요성

### 6.3 커뮤니티 톤
- 친절하고 포용적
- 경쟁과 우애의 균형
- 콘텐츠 크리에이터 지원

## 8. 프로젝트 제약 조건

### 8.1 기술적 제약

- **초기 카드 세트**: 200장 (공개 가능한 캐릭터 기반)
- **1차 확장팩**: 분기별 50장
- **디지털 플랫폼**: 웹 우선, 모바일 2024년 내 지원

### 8.2 비즈니스 제약

- **라이센스 제약**: 주술회전 정식 IP 라이센스 필수
- **지역 제약**: 한국, 일본, 미국 우선 출시
- **성인용 콘텐츠 금지**: 모든 카드는 12세 이상 관람가

### 8.3 설계 제약

- **카드 비용 범위**: 0~10
- **카드당 텍스트**: 최대 3줄 (초보자 이해도 고려)
- **카드 유형**: 최대 5가지
- **게임 길이**: 절대 20분을 초과하지 않음
