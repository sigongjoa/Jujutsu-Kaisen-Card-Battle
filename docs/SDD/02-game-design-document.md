## 1. 게임 개요

### 1.1 게임 정의

**Jujutsu Kaisen Card Battle**은 주술회전 세계관을 배경으로 하는 2인 대전 TCG입니다.

- **게임 시간**: 8~12분
- **플레이어 수**: 2명
- **난이도**: 초급~상급
- **복잡도**: 중간 (5/10)

### 1.2 게임의 목표

**상대 플레이어의 HP를 0으로 만드는 것**

- 각 플레이어는 20 HP로 시작
- 주술 공격으로 상대방에게 손상 입히기
- 가장 먼저 상대를 무력화하는 플레이어가 승리

## 2. 핵심 메커니즘 (Core Mechanics)

### 2.1 자원 시스템 (Resource System): 저주력 (Cursed Energy)

#### 2.1.1 저주력의 개념

저주력(Cursed Energy, CE)은 주술을 사용하기 위한 필수 자원입니다.

```
┌─────────────────────────────────┐
│   Cursed Energy (저주력)         │
├─────────────────────────────────┤
│  • 매 턴 시작에 자동 증가       │
│  • 1턴마다 +1 증가 (최대 10)    │
│  • 카드로 회복 불가능            │
│  • 턴이 끝나면 미사용분 소멸     │
│  • 스켈레톤: 초반부터 후반까지  │
│    점진적으로 강한 주술 사용   │
└─────────────────────────────────┘
```

#### 2.1.2 저주력 추이 예시

| 턴 | 누적 저주력 | 사용 가능 | 사용 후 | 비고 |
|----|---------|----|-------|------|
| 1 | 1 | 1 | 0 | 약한 주술 1개 |
| 2 | 3 | 3 | 0 | 약한 주술 2개 또는 중간 주술 1개 |
| 3 | 6 | 6 | 0 | 강력한 주술 1개 + 약한 주술 2개 |
| 4 | 10 | 10 | 0 | 최강 주술 사용 가능 |

### 2.2 카드 유형 분류 (Card Types)

```
┌──────────────────────────────────────────┐
│        카드 타입 분류                    │
├──────────────────────────────────────────┤
│                                          │
│  1. 저주술사 (Curse User / Jujutsu)    │
│     └─ HP, 능력, 주술이 있는 메인 카드 │
│                                          │
│  2. 주술 (Cursed Technique / Skill)    │
│     └─ 일회용 효과 또는 지속 효과       │
│                                          │
│  3. 저주물 (Cursed Object / Item)      │
│     └─ 연결되는 지속물, 장비 같은 것   │
│                                          │
│  4. 사건 (Event / Incident)            │
│     └─ 일회용, 양쪽에 영향을 미는 효과 │
│                                          │
│  5. 반응 (Response / Trap)            │
│     └─ 상대턴에 발동하는 방어/카운터    │
│                                          │
└──────────────────────────────────────────┘
```

#### 2.2.1 카드 유형별 특징

| 유형 | 비용 | 효과 | 제거 방법 |
|------|------|------|---------|
| **저주술사** | 2~8 | 공격력, 체력, 능력 | 전투로만 제거 |
| **주술** | 1~5 | 일회용 또는 지속 | 파괴 주문으로 제거 |
| **저주물** | 2~6 | 지속 효과 | 파괴 주문으로 제거 |
| **사건** | 2~4 | 일회용 글로벌 효과 | 자동 소멸 |
| **반응** | 1~3 | 조건부 반응 | 발동 후 소멸 |

### 2.3 게임 영역 (Zones)

```
┌────────────────────────────────────────────┐
│           Player A (You)                   │
├────────────────────────────────────────────┤
│                                            │
│  [Your Hand]  [Deck]                       │
│  (카드 위치   (남은 카드)                   │
│   비공개)                                  │
│                                            │
│  ┌──────────────────────────────────────┐ │
│  │ Battlefield (필드)                    │ │
│  │ • 저주술사 최대 3명                   │ │
│  │ • 주술/저주물 최대 5개                │ │
│  └──────────────────────────────────────┘ │
│                                            │
│  [Graveyard]  [Banished Zone]             │
│  (무덤)        (제외 구역)                 │
│                                            │
└────────────────────────────────────────────┘

┌────────────────────────────────────────────┐
│           Player B (Opponent)              │
├────────────────────────────────────────────┤
│ (위와 동일하지만 상대 시점에서)            │
└────────────────────────────────────────────┘
```

#### 2.3.1 각 영역의 정의

| 영역 | 공개/비공개 | 용도 | 최대 크기 |
|------|-----------|------|---------|
| **Hand** | 비공개 | 플레이할 카드 보유 | 무제한 |
| **Deck** | 비공개 | 미사용 카드 저장소 | 40~60장 |
| **Battlefield** | 공개 | 활성화된 카드 전시 | 저주술사 3명, 주술/저주물 5개 |
| **Graveyard** | 공개 | 사용/파괴된 카드 | 무제한 |
| **Banished Zone** | 공개 | 영구 제거된 카드 | 무제한 |

## 3. 턴 구조 (Turn Structure)

### 3.1 한 턴의 흐름도

```
┌──────────────────────────────────────┐
│  턴 시작 (Turn Start)                │
├──────────────────────────────────────┤
│  1. 재충전 단계 (Recharge Phase)    │
│     └─ 저주력 +1 증가, 최대값 +1    │
│     └─ 손상 카드 복구 체크          │
│                                      │
│  2. 카드 뽑기 (Draw Phase)          │
│     └─ 덱에서 카드 1장 뽑기        │
│     └─ 덱에 카드가 없으면 게임 오버│
│                                      │
│  3. 메인 단계 A (Main Phase A)      │
│     └─ 저주술사/주술/저주물 플레이 │
│     └─ 능력 활성화 가능              │
│                                      │
│  4. 전투 단계 (Battle Phase)        │
│     └─ 저주술사로 공격 선언         │
│     └─ 상대 방어 또는 전투          │
│                                      │
│  5. 메인 단계 B (Main Phase B)      │
│     └─ 추가 플레이 가능 (선택)     │
│                                      │
│  6. 종료 단계 (End Phase)           │
│     └─ 손패 5장 초과 시 버리기     │
│     └─ 지속 효과 처리 종료          │
│                                      │
└──────────────────────────────────────┘
        ↓
   상대 플레이어의 턴 시작
```

### 3.2 턴 단계별 상세 설명

#### 3.2.1 재충전 단계 (Recharge Phase)

```
Step 1: 저주력 증가
├─ 현재 저주력: 3
├─ 증가량: +1
└─ 새로운 저주력: 4 (최대값도 4로 증가)

Step 2: 지속 효과 처리
├─ "턴 시작 시" 효과 처리
├─ 손상된 카드 복구 체크
└─ 상태 이상 업데이트 (예: 중독)
```

#### 3.2.2 카드 뽑기 (Draw Phase)

```
• 일반적으로: 카드 1장 뽑기
• 특수 효과로: 추가 카드 뽑기 가능
• 무덤에 카드만 남은 상태에서 뽑을 카드가 없으면?
  └─ 즉시 패배 (손상이 아님)
```

#### 3.2.3 메인 단계 A & B (Main Phases)

```
각 메인 단계에서 가능한 행동:

1. 저주술사 소환 (한 턴 1명만 가능)
   ├─ 비용만큼 저주력 소비
   └─ 필드에 배치 (최대 3명)

2. 주술/저주물 플레이 (횟수 제한 없음)
   ├─ 비용만큼 저주력 소비
   └─ 해당 효과 즉시 처리

3. 활성화 능력 사용
   ├─ 저주력 또는 기타 비용 소비
   └─ 필드의 카드 능력 발동

4. 특수 액션
   ├─ 패스 (더 이상 할 행동이 없음)
   └─ 우선권 포기
```

#### 3.2.4 전투 단계 (Battle Phase)

```
공격자 선언 (Active Player)
├─ 필드의 저주술사 1명 선택
├─ 상대 플레이어 또는 상대 저주술사 선택
└─ 공격 선언

방어자 선택 (Passive Player)
├─ 전투 중단하지 않음 (기본 처리)
├─ 저주술사로 블록 (전투 진행)
└─ 반응 카드 사용 (공격 취소 등)

전투 처리
├─ 쌍방 손상 계산
├─ 생존 여부 결정
└─ 전투 상처/상태 이상 적용
```

### 3.3 우선권 (Priority) 시스템

```
┌────────────────────────────────────────┐
│  우선권 시스템 (Simplified)            │
├────────────────────────────────────────┤
│                                        │
│  기본 원칙: 활성화된 가장 최근        │
│  카드부터 처리 (Last-In-First-Out)   │
│                                        │
│  예시:                                 │
│  1. 플레이어 A가 카드 X 플레이        │
│  2. 플레이어 B가 카드 Y로 대응        │
│  3. 플레이어 A가 카드 Z로 재대응     │
│  ∴ 처리 순서: Z → Y → X             │
│                                        │
└────────────────────────────────────────┘
```

## 5. 전투 메커니즘 (Combat)

### 5.1 전투 흐름

```
Phase 1: 공격 선언
├─ 공격자가 저주술사 선택
├─ 공격 대상 선택 (플레이어 또는 저주술사)
└─ "공격한다"라고 선언

Phase 2: 방어 결정
├─ Option A: 블록 안 함 → 손상 직접 입음
├─ Option B: 저주술사로 블록
│  └─ 전투 발생 (양쪽 손상 계산)
└─ Option C: 반응 카드 사용 → 전투 취소 등

Phase 3: 전투 해결 (블록 선택 시)
├─ 공격자의 공격력 = 손상
├─ 방어자의 공격력 = 공격자 손상
└─ 동시에 손상 입음

Phase 4: 전투 결과 처리
├─ HP가 0 이하 → 저주술사 무력화
├─ 특수 효과 처리 (트리거 능력 등)
└─ 전투 상처 또는 상태 이상 적용
```

### 5.2 예제: 전투 시뮬레이션

```
상황:
- 플레이어 A: 공격력 3, 체력 5인 저주술사 사용
- 플레이어 B: 공격력 2, 체력 4인 저주술사 블록 선택

처리:
1. A의 저주술사가 B의 저주술사에게 3 손상
   └─ B의 저주술사: 4 - 3 = 1 HP 남음

2. B의 저주술사가 A의 저주술사에게 2 손상
   └─ A의 저주술사: 5 - 2 = 3 HP 남음

결과:
- A의 저주술사: 3 HP (생존)
- B의 저주술사: 1 HP (생존, 매우 약화됨)
```

### 5.3 특수 전투 규칙

| 규칙 | 설명 |
|------|------|
| **관통 (Piercing)** | 초과 손상이 플레이어에게 직접 입혀짐 |
| **회피 (Evasion)** | 이 저주술사는 블록당할 수 없음 |
| **불굴 (Indestructible)** | 전투 손상을 받지 않음 |
| **생명력 흡수 (Lifelink)** | 입힌 손상만큼 HP 회복 |

## 6. 게임 상태 및 종료 조건 (Win/Loss Conditions)

### 6.1 게임 종료 조건

| 조건 | 결과 |
|------|------|

## 7. 키워드 및 능력 용어집

### 7.1 공통 키워드

| 키워드 | 비용 | 효과 |
|--------|------|------|
| **드로우** | - | 덱에서 카드 1장 뽑기 |
| **도발** | - | 이 카드가 플레이되면 상대 저주술사로 공격해야 함 |
| **회피** | - | 이 저주술사는 블록당할 수 없음 |
| **생명력 흡수** | - | 이 저주술사가 입힌 손상만큼 HP 회복 |
| **관통** | - | 블록한 저주술사를 무력화하면 초과 손상이 플레이어에게 입혀짐 |
| **불굴** | - | 전투 손상을 받지 않음 |

### 7.2 상태 이상 (Status Effects)

| 상태 | 영향 | 해제 |
|------|------|------|
| **중독 (Poison)** | 턴 종료 시 1 손상 | 특수 카드로만 해제 |
| **약화 (Weakened)** | 공격력 50% 감소 | 턴 종료 시 자동 해제 |
| **마비 (Paralyzed)** | 공격할 수 없음 | 턴 종료 시 자동 해제 |

### 8.2 주술 카드 예시


## 9. 밸런싱 철학

### 9.1 핵심 원칙

1. **비용과 효율의 균형**
   - 같은 비용의 카드는 비슷한 "가치"를 가져야 함
   - 특수 능력 있는 카드는 기본 능력이 약할 수 있음

2. **다양성 우선**
   - 한 가지 덱 타입만 우수하면 안 됨
   - 최소 5가지 이상의 경합 가능한 덱 타입 존재

3. **초보자-고급자 간 진입장벽 최소화**
   - 1~2코스트 카드들도 경쟁력 있어야 함
   - 높은 비용 카드가 절대적으로 우수하면 안 됨

### 10.1 확장팩 설계 원칙
