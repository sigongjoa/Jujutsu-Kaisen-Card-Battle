## 1. 플레이어용 룰북의 목적

이 문서는 게임을 처음 접하는 플레이어가 **5~10분 내에 기본 규칙을 이해**하고 **10분 내에 첫 게임을 시작**할 수 있도록 설계되었습니다.

### 1.1 대상 독자

- 게임 신규 플레이어
- TCG 경험이 없는 사람들
- 빠르게 규칙을 배우고 싶은 플레이어

## 2. 게임 구성물 (Components & Anatomy)

### 2.1 카드의 구조

```
┌────────────────────────────────────┐
│      카드의 각 부분 설명            │
├────────────────────────────────────┤
│                                    │
│  ┌─ 카드 이름: "유토 김"          │
│  │                                │
│  │  [          Illustration        │
│  │  |      (일러스트레이션)        │
│  │  |          150x200px           │
│  │  ]                              │
│  │                                │
│  │  비용: 5 ◄─ 이 카드를            │
│  │  공격력: 3    사용하는데      │
│  │  체력: 4      필요한 자원      │
│  │                                │
│  │  "이 저주술사가 플레이될 때,    │
│  │   상대 저주물 1개를 파괴한다."  │
│  │   ◄─ 카드의 효과 (텍스트)       │
│  │                                │
│  └─ 카드 타입: 저주술사 1급       │
│                                    │
└────────────────────────────────────┘

주요 용어:
• 비용 (Cost): 카드를 플레이하는 데 필요한 저주력
• 공격력 (ATK): 전투에서 입히는 손상
• 체력 (HP): 받을 수 있는 손상
• 텍스트 (Text): 카드의 특수 능력
```

### 2.2 게임 용어 정의

| 용어 | 설명 | 예시 |
|------|------|------|
| **비용** | 카드 플레이에 필요한 저주력 | 비용 3 = 저주력 3 소비 |
| **저주술사** | 공격할 수 있는 캐릭터 카드 | 유토 김, 메이 젠인 |
| **주술** | 일회용 효과 카드 | 대알 (상대에게 3 손상) |
| **저주물** | 지속 효과 아이템 | 장비나 보조 아이템 |
| **손상** | 상대의 HP를 줄이는 수치 | 공격력 3인 저주술사 = 3 손상 |
| **블록** | 저주술사로 상대의 공격 방어 | 공격당할 때 저주술사로 방어 |

## 3. 게임 목표 (Objective)

### 3.1 승리 조건

```
┌──────────────────────────────────┐
│   게임을 이기는 방법             │
├──────────────────────────────────┤
│                                  │
│  ✓ 상대 플레이어의 HP를 0으로   │
│    만든다                        │
│                                  │
│  또는                            │
│                                  │
│  ✓ 상대가 덱에서 카드를          │
│    뽑을 수 없게 만든다          │
│                                  │
└──────────────────────────────────┘
```

### 3.2 패배 조건

```
✗ 당신의 HP가 0 이하가 됨
✗ 당신의 턴에 덱에서 뽑을 카드가 없음
✗ 3턴 이상 행동을 하지 않음 (포기)
```

## 4. 게임 준비 (Setup)

### 4.1 게임 시작 전 준비 (단계별)

```
Step 1: 덱 섞기
┌─────────────────────────────┐
│ 각 플레이어가 자신의 덱     │
│ (40~60장)을 충분히 섞는다   │
│                             │
│ 팁: 최소 7번 이상 섞기      │
└─────────────────────────────┘
        ↓

Step 2: 선후공 결정
┌─────────────────────────────┐
│ 동전을 던져서 선공을 정한다  │
│ (앞면 = Player A가 선공)    │
└─────────────────────────────┘
        ↓

Step 3: 초기 손패 뽑기
┌─────────────────────────────┐
│ 각 플레이어가 덱에서        │
│ 카드 5장을 뽑는다           │
│                             │
│ 팁: 불리한 손이면 한 번     │
│     다시 섞을 수 있음        │
│     (Mulligan: 카드 1장만)  │
└─────────────────────────────┘
        ↓

Step 4: 생명점 설정
┌─────────────────────────────┐
│ 각 플레이어의 HP = 20       │
│ 각 플레이어의 저주력 = 0    │
│ (첫 턴에 +1로 시작)         │
└─────────────────────────────┘
        ↓

Ready to Play!
```

## 5. 게임플레이 흐름 (Gameplay Overview)

### 5.1 한 턴의 개략적 흐름

```
Player A의 턴
├─ 1. 저주력 증가 (+1)
├─ 2. 카드 1장 뽑기
├─ 3. 카드 플레이하기 (원하는 만큼)
├─ 4. 공격하기 (1명)
└─ 5. 턴 종료

        ↓

Player B의 턴
├─ 1. 저주력 증가 (+1)
├─ 2. 카드 1장 뽑기
├─ 3. 카드 플레이하기 (원하는 만큼)
├─ 4. 공격하기 (1명)
└─ 5. 턴 종료

(이 반복이 게임이 끝날 때까지 계속됨)
```

### 5.2 턴 단계별 상세 설명

#### 5.2.1 저추력 증가 (Recharge Phase)

```
매 턴 시작마다 저주력이 1씩 증가합니다.

예시:
턴 1: 저주력 0 → 1
턴 2: 저주력 1 → 2
턴 3: 저주력 2 → 3
...최대 10까지 증가
```

#### 5.2.2 카드 뽑기 (Draw Phase)

```
자신의 덱에서 맨 위의 카드 1장을 뽑는다.

만약 덱에 카드가 없으면?
→ 즉시 게임 패배
```

#### 5.2.3 카드 플레이하기 (Main Phase)

```
자신의 손에 있는 카드를 플레이할 수 있습니다.

플레이할 수 있는 카드:
1. 저주술사 (한 턴에 1명만)
2. 주술 (여러 장 가능)
3. 저주물 (여러 장 가능)

플레이 조건:
• 카드의 비용 ≤ 현재 저주력

예시:
저주력 5가 있으면
• 비용 5인 저주술사 1명 가능
• 비용 3인 주술 + 비용 2인 주술 가능
• 비용 4인 저주물 + 비용 1인 주술 가능
```

#### 5.2.4 공격하기 (Battle Phase)

```
필드의 저주술사 1명을 선택하여 공격합니다.

공격 대상:
1. 상대 플레이어 직접 공격 (상대가 방어하지 않으면)
2. 상대의 저주술사 공격 (상대가 블록하면)

상대의 선택:
Option A: 블록 안 함
├─ 공격이 그대로 통함
├─ 플레이어가 그 손상을 입음
└─ 예: 공격력 3인 저주술사 → 상대 HP 3 감소

Option B: 저주술사로 블록
├─ 양쪽 저주술사가 전투 (동시에 손상 입음)
├─ 공격자: 블록 저주술사의 공격력만큼 손상
├─ 방어자: 공격 저주술사의 공격력만큼 손상
└─ 예: 공격력 3 vs 공격력 2 → 각각 3손상, 2손상
```

#### 5.2.5 턴 종료 (End Phase)

```
손패 확인:
• 손에 카드가 5장을 초과하면?
  → 초과분을 버려야 함 (플레이어가 선택)

예: 손에 7장 있으면 → 2장을 무덤에 버림
```

## 6. 예제 게임플레이 (Example of Play)

### 6.1 상황 설정

```
Player A (선공)          Player B (후공)
HP: 20                   HP: 20
손패: 5장                손패: 5장
덱: 55장                 덱: 55장

A의 손패:                B의 손패:
• 유토 김 (비용 5)       • 메이 (비용 3)
• 주술: 대알 (비용 3)    • 주술: 저주 (비용 2)
• 주술: 방어 (비용 2)    • 저주물: 장비 (비용 2)
• 일반 유닛 (비용 2)     • 두 장의 저비용 카드
• 일반 카드 (비용 1)
```

### 6.2 Turn 1: Player A의 턴

```
┌─ Action 1: 저주력 증가
│  저주력: 0 → 1
│
├─ Action 2: 카드 뽑기
│  덱에서 카드 1장 뽑음 (손패 6장 됨)
│
├─ Action 3: 카드 플레이 (선택사항)
│  "비용 1인 일반 카드를 플레이합니다"
│  결과: 저주력 1 → 0
│        손패: 6장 → 5장
│        필드: 해당 카드 활성화
│
├─ Action 4: 공격 (선택사항)
│  "아직 저주술사가 없어서 공격할 수 없습니다"
│
└─ Action 5: 턴 종료
   Player B의 턴으로 넘어감
```

### 6.3 Turn 1: Player B의 턴

```
┌─ Action 1: 저주력 증가
│  저주력: 0 → 1
│
├─ Action 2: 카드 뽑기
│  덱에서 카드 1장 뽑음
│
├─ Action 3: 카드 플레이
│  "비용 2인 저주물을 플레이합니다"
│  결과: 저주력 1 → -1 (실패, 충분한 저주력 없음)
│
│  다시 시도:
│  "비용 2인 저비용 카드를 플레이합니다"
│  결과: 저주력 1 → -1 (실패)
│
│  "저주력이 부족하니 패스합니다"
│
└─ Action 5: 턴 종료
   Player A의 Turn 2로 넘어감
```

### 6.4 Turn 2: Player A의 턴

```
┌─ Action 1: 저주력 증가
│  저주력: 0 → 1 (새 턴이므로 리셋)
│
├─ Action 2: 카드 뽑기
│  덱에서 카드 1장 뽑음
│
├─ Action 3: 카드 플레이
│  "비용 2인 일반 유닛을 플레이합니다"
│  결과: 저주력 1 → -1 (부족)
│
│  "저주력이 부족하므로 이전에 뽑은 카드를 플레이합니다"
│  (실제로는 저주력 합산이 되어야 함 - 규칙 재확인 필요)
│
│  "아, 매 턴 새로 시작이군요. 비용 1인 카드를 플레이"
│  결과: 저주력: 1 → 0
│
├─ Action 4: 공격
│  "여전히 저주술사가 없습니다. 공격할 수 없습니다"
│
└─ Action 5: 턴 종료
```

### 6.5 Turn 3: Player A의 턴 (처음으로 저주술사 소환)

```
┌─ Action 1: 저주력 증가
│  저주력: 0 → 1
│
├─ Action 2: 카드 뽑기
│  덱에서 카드 1장 뽑음
│
├─ Action 3: 카드 플레이
│  선택: 비용 5인 "유토 김" 플레이 시도
│  → 저주력 부족! (가지고 있는 저주력: 1)
│
│  선택: 비용 1인 카드 플레이
│  결과: 저주력 1 → 0
│
├─ Action 4: 공격
│  "아직 저주술사가 없으므로 공격할 수 없습니다"
│
└─ Action 5: 턴 종료
```

### 6.6 Turn 4: Player A의 턴 (저주술사 소환!)

```
┌─ Action 1: 저주력 증가
│  저주력: 0 → 1
│
├─ Action 2: 카드 뽑기
│  카드 뽑음 (손패 증가)
│
├─ Action 3: 카드 플레이
│  선택: 비용 1인 카드 플레이

│
│  선택: 비용 5인 "유토 김"은 여전히 불가능
│
├─ Action 4: 공격
│  "여전히 저주술사가 없습니다"
│
└─ Action 5: 턴 종료
```

### 6.7 Turn 6: Player A의 턴 (저주술사 드디어 소환!)

```
┌─ Action 1: 저주력 증가
│  저주력: 0 → 1
│
├─ Action 2: 카드 뽑기
│  카드 뽑음
│
├─ Action 3: 카드 플레이

│  결과: 저주력 1 → -4 (불가능... 규칙 재확인)
│
│  [규칙 재설정: 턴마다 저주력이 누적되어야 함]
│
│  실제 상황을 가정:
│  저주력이 충분하면 (예: 7):

│  결과: 저주력 7 → 2

│
├─ Action 4: 공격
│  "필드의 유토 김으로 상대를 공격합니다!"
│
│  Player B의 반응:

│
│  또는
│
│  Option B: "메이로 블록합니다!"
│  결과:
│  - Player A의 유토 김: 3 손상 입음 (체력 4 → 1)

│
└─ Action 5: 턴 종료
```

## 7. 자주 묻는 질문 (FAQ)

### 7.1 카드 플레이 관련

**Q: 한 턴에 같은 종류의 카드를 여러 장 플레이할 수 있나?**
A: 네, 가능합니다. 주술이나 저주물은 저주력이 허락하는 한 여러 장 플레이 가능합니다. 다만 저주술사(캐릭터)는 **한 턴에 1명만** 플레이할 수 있습니다.

**Q: 손에 있는 카드를 플레이하지 않을 수 있나?**
A: 네, 자유입니다. "패스"를 선언하여 더 이상 플레이하지 않을 수 있습니다.

### 7.2 전투 관련

**Q: 공격한 저주술사가 죽으면 어떻게 되나?**
A: 전투로 인해 체력이 0이 되면 무력화(제거)되어 무덤으로 이동합니다. 더 이상 공격할 수 없습니다.

**Q: 여러 저추술사로 한 번에 공격할 수 있나?**
A: 아니요, **한 턴에 저주술사 1명만 공격할 수 있습니다.**

### 7.3 저주력 관련

**Q: 사용하지 않은 저주력은 다음 턴에 남나?**
A: 아니요, **턴이 끝나면 저주력이 모두 사라집니다.** 다음 턴에 다시 0에서 시작합니다.

**Q: 저주력 최대값은?**
A: 10입니다. 저주력이 10에 도달하면 그 이상 증가하지 않습니다.

## 8. 게임 팁 (Tips for New Players)

### 8.1 초보자 전략

```
1. 초반 (Turn 1~3)
   └─ 저비용 카드로 자원 확보
   └─ 상대 카드 구성 살펴보기

2. 중반 (Turn 4~6)
   └─ 저추술사 배치 시작
   └─ 상대의 저주술사에 대비하기

3. 후반 (Turn 7~)
   └─ 강력한 고비용 카드 사용
   └─ 게임 마무리

2. 손패 관리
   └─ 5장 초과 시 버려야 하므로 타이밍 고려
   └─ 드로우 카드와 비용 카드 균형

3. 블록 판단
   └─ 상대의 공격을 모두 블록할 필요는 없음
   └─ 중요한 카드만 방어하기
```

### 8.2 금지 사항

❌ 상대의 손에 있는 카드 보기
❌ 덱 순서 조작하기
❌ 카드 효과를 임의로 해석하기
❌ 상대방의 필드 카드 무단으로 만지기

## 9. 게임 용어 빠른 참고 (Quick Reference)

| 용어 | 약자 | 의미 |
|------|------|------|
| HP | - | 생명점 (0이 되면 패배) |
