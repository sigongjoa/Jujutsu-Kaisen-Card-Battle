## 4. 계약 내용

### 4.1 작업 범위 (Scope of Work)

```
발주자는 수주자에게 다음의 작업을 의뢰합니다:

1. 작업 내용
   - 작업명: [예: "유토 김 카드 일러스트"]
   - 작업 상세: [구체적인 사양]
   - 파일 포맷: [PSD, TIFF 등]
   - 해상도: [300 DPI 등]

2. 납기일
   - 스케치 완성: [날짜]
   - 채색 완성: [날짜]
   - 최종 납품: [날짜]

3. 수정 횟수
   - 무제한 수정 (단, 초기 기획 범위 내)
   - 범위 외 요청 시 추가 비용 발생

4. 개수
   - 총 [X]개 작품
```

### 4.2 대가 (Compensation)

```
1. 보수
   - 총액: [금액] 원
   - 지불 방식: [일시불 / 분할금]
   - 지불 일정:
     * 계약 시: [금액] (선금)
     * 스케치 승인 시: [금액]
     * 최종 납품 시: [금액]

2. 추가 비용
   - 범위 외 요청: [금액]/회
   - 납기 지연 시: 일일 [금액] 감액

3. 세금
   - [수주자]는 개인사업자로서 세금 처리
   - 원천징수세 [XX%] 공제 후 지급
```

### 4.3 저작권 및 소유권 (Intellectual Property Rights)

**매우 중요한 조항입니다.**

```
1. 저작권 귀속
   제작된 모든 결과물(이하 "저작물")의 저작권 및
   저작인접권은 발주자에게 배타적으로 귀속됩니다.

2. 2차 저작물 작성권 (Derivative Rights)
   - 수주자는 저작물을 바탕으로 2차 저작물을
     작성할 권리가 없습니다.
   - 예: 카드 일러스트를 바탕으로 그림책,
        애니메이션, 스핀오프 게임 제작 등

3. 이용 범위
   발주자는 다음 목적으로 저작물을 자유롭게 이용할 수 있습니다:
   ✓ 게임 카드 인쇄 및 판매
   ✓ 디지털 게임 클라이언트 표시
   ✓ 게임 관련 마케팅 자료
   ✓ 게임 공식 웹사이트 게재
   ✓ 라이센싱 파트너에게 제공
   ✓ 향후 모든 미디어 미디어화 (영상화, 소설화 등)

4. 크레딧 표기
   - 게임 카드: 아티스트명 기재 (크레딧)
   - 디지털 버전: 게임 내 아티스트 목록에 기재
   - 공식 홈페이지: 아티스트 페이지 링크 제공
   - 라이센싱 자료: 원저작자 표기 유지
```

### 4.4 기밀 유지 (Confidentiality)

```
1. 기밀 정보의 범위
   - 게임 규칙, 카드 능력 (공개 전)
   - 게임 개발 계획
   - 수익성과 판매 계획
   - 마케팅 전략

2. 의무
   수주자는 다음을 약속합니다:
   - 기밀 정보를 제3자에게 공개하지 않음
   - 소셜 미디어에 작업 과정 사진/영상 게재 금지
   - 계약 종료 후 [3년/5년] 동안 기밀 유지
   - 요청 시 작업 파일 및 자료 폐기

3. 예외
   다음의 경우는 기밀 의무가 면제됩니다:
   - 법원의 명령
   - 정부 기관의 요청
   - 발주자가 공식 공개한 정보
```

### 4.5 품질 기준 (Quality Standards)

```
1. 기술적 사양
   - 해상도: 300 DPI 이상
   - 색상 모드: RGB (최종 CMYK로 변환)
   - 파일 포맷: PSD (레이어 유지), TIFF, JPG
   - 파일 크기: [최대 용량]

2. 디자인 기준
   - 아트 바이블 준수 (세력 색상, 스타일 등)
   - 카드 레이아웃 안전 영역 준수
   - 텍스트 가독성 확보
   - 원작 IP 충실성

3. 검수 기준
   발주자가 승인하지 않는 경우:
   - 아티스트는 지적사항 수정
   - 무제한 수정 (기획 범위 내)
   - 범위 외 요청: 추가 비용
```

### 4.6 제작 일정 및 지연 (Timeline & Delays)

```
1. 작업 일정
   [Gantt 차트 또는 마일스톤 기입]

2. 지연 시
   - 원인이 아티스트 책임: 일일 [금액] 감액
   - 원인이 발주자 책임: 연장료 지급
   - 긴급 완성 요청: 추가 비용 [금액]

3. 불가항력
   천재지변, 전쟁, 대규모 감염병 등의 경우
   기한 연장 가능
```

### 4.7 수정 및 피드백 (Revisions & Feedback)

```
1. 수정 절차
   스케치 → 채색 → 최종 완성
   각 단계에서 발주자 승인 필요

2. 수정 횟수
   각 단계별 무제한 수정 (기획 범위 내)

3. 범위 외 요청 (Out of Scope)
   예: "카드 배경을 완전히 다른 장면으로"
   이 경우 추가 비용 협의

4. 최종 승인
   발주자가 "최종 승인"을 한 후에만
   결재금 지급 및 저작권 이전
```

### 4.8 지적재산권 보증 (Warranty of IP Rights)

```
수주자는 다음을 보증합니다:

1. 저작물은 자신이 독립적으로 창작한 것
2. 저작물은 제3자의 지적재산권을 침해하지 않음
3. 저작물에는 저작권이 있는 외부 자료가 미포함
4. 저작물은 모두 규정된 범위 내의 원작과 일관성 있음

만약 침해 사실이 밝혀지면:
- 수주자가 모든 법적/금전적 책임 부담
- 발주자는 손해배상청구 권리 보유
```

### 4.9 비밀유지 및 소멸 (Confidentiality Termination)

```
계약 종료 후:
- 수주자는 모든 작업 자료/초안 폐기
- 백업 파일도 삭제 (증명 가능해야 함)
- 개인 포트폴리오에도 미공개 상태 유지
```

### 4.10 계약 종료 (Termination)

```
1. 상호 합의 종료
   양쪽이 합의하면 언제든 종료 가능
   - 완료한 작업: 대금 지급
   - 미완료 작업: 진행률 [XX%]만큼 지급

2. 발주자의 일방 종료
   사유: 아티스트의 계약 위반 (예: 납기 지연 30일+)
   - 완료한 작업까지만 대금 지급
   - 미완료 작업: 대금 미지급

3. 종료 후 의무
   - 모든 자료 회수 및 폐기
   - 기밀유지 의무 계속 유효
   - 저작권은 발주자에게 유지
```

  (촬영권, 프라이버시 침해 등)
```

### 6.2 독립 계약자 (Independent Contractor)

```
아티스트는 [발주사]의 직원이 아닌 독립 계약자입니다.
따라서:
- 고용 관계 없음
- 산재 보험 미적용
- 퇴직금 없음
- 4대보험 가입 (아티스트 책임)
```

### 6.3 발표 권리 (Right of Attribution)

```
발주자는 캠페인, 기사, 포트폴리오에서
아티스트를 저작자로 명시할 수 있습니다.

아티스트는 카드의 "Artist: [이름]" 크레딧 기재를
요청할 수 있습니다. (발주자는 합리적인 범위에서 수용)
```

## 9. 법적 유의사항

### ⚠️ 중요 경고

1. **법률 검토 필수**
   - 이 템플릿은 일반적 참고용입니다

4. **일반적 문제**

   - 해지 시 미수금 처리 규정
